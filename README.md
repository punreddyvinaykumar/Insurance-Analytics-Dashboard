# Insurance Claims & Policy Performance Analytics Dashboard

## Project Overview

This project analyzes insurance policy and claims data using SQL Server and Power BI.

The objective is to provide business stakeholders with insights into:

- Premium Revenue
- Coverage Exposure
- Claims Activity
- Policy Performance
- Customer Demographics

The data was sourced from Excel, loaded into SQL Server, transformed using Power Query, and visualized in Power BI.

## Dashboard Insights

### Policy Performance

Travel Insurance generated the highest premium revenue among all policy categories.

### Claims Analysis

Adults contributed the highest claim amount compared to other age groups.

### Policy Monitoring

Inactive policies represented a significant portion of the overall policy portfolio.

### Business Value

The dashboard enables insurance stakeholders to monitor policy performance, claims activity, customer demographics, and revenue trends through a centralized reporting solution.
