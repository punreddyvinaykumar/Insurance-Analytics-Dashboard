# Insurance Claims & Policy Performance Analytics Dashboard

## Project Overview

This project analyzes insurance policy and claims data using SQL Server and Power BI.

The objective is to provide business stakeholders with insights into:

- Premium Revenue
- Coverage Exposure
- Claims Activity
- Policy Performance
- Customer Demographics

The data was sourced from Excel, loaded into SQL Server, transformed using Power Query, and visualized in Power BI.
